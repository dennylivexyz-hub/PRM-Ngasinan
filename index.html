Konfigurasi Google Sheets

Hubungkan dengan Google Sheets agar data anggota, kas, dan laporan tersimpan & terupdate otomatis. Ikuti langkah berikut:

1. Buka Google Sheets → Buat spreadsheet baru
2. Klik Extensions → Apps Script
3. Paste kode Web App yang tersedia (hubungi admin)
4. Deploy → "New deployment" → pilih Web app
5. Salin URL di bawah ke kolom ini
